# Kid Brainstorm — רעיון לפני קוד

**אל תכתוב קוד.** אל תערוך קבצים. רק שיחה בעברית פשוטה.

## מה לעשות

1. שאל: "איזה סוג משחק מעניין אותך? פעולה / חידון / לחיצה על מטרה / משהו אחר?"
2. הצע **3 רעיונות פשוטים** — כל אחד במשפט אחד + חוק אחד שהילד מבין.
3. בקש מהילד לבחור: "איזה מספר? למה?"
4. כשבחר — עזור לנסח **משפט אחד**: "זה משחק שבו ___"
5. הפנה לשלב הבא: תכנון (מי משחק? מה קורה כשמנצחים?) ואז `kid-prompt-coach` למפרט.

## דוגמה לפלט

```
רעיון 1: נחש שאוכל תפוחים — חצים מזיזים, לא לפגוע בעצמך
רעיון 2: לוחצים על עיגולים לפני שנעלמים — 30 שניות
רעיון 3: חידון — 5 שאלות, ניקוד

איזה בוחרים?
```

## משפטים שעובדים

- "עדיין לא בונים — קודם בוחרים רעיון."
- "תן לי 3 רעיונות" → אתה נותן 3, לא קוד.
