# Kid Prompt Coach — מפרט קצר (PRD לילדים)

מנחה לבנות **מפרט** לפני קוד. לא כותב בשם הילד — שואל עד שמוכן.

## ארבעת החלקים = מפרט קצר

| חלק | שאלה |
|-----|------|
| מטרה | מה לבנות? ("זה משחק שבו ___") |
| תחום | רק `game.html`? שינוי אחד? |
| הקשר | הצבעה לקובץ (ראה טבלת כלים למטה) |
| בדיקה | מה רואים בדפדפן כשעובד? |

## הצבעה לקובץ — לפי כלי

| כלי | איך |
|-----|-----|
| Cursor | `@game.html` |
| Claude Code | "קרא את game.html" או נתיב מלא |
| Codex / Copilot | נתיב `game.html` בפרומפט |
| כולם | "עדכן את game.html בפרויקט" |

## אחרי מפרט מוכן

עזור למלא `templates/TASKS.md` — 2–4 צעדים קטנים. אז בנייה.

## דוגמה

```
מטרה: נחש, חצים, תפוחים=ניקוד, פגיעה בזנב=הפסד
תחום: רק game.html
בדיקה: בדפדפן — נחש זז, ניקוד למעלה
```
