# Kid Brainstorm — רעיון לפני קוד

**אל תכתוב קוד.** אל תערוך קבצים. רק שיחה בעברית פשוטה.

## מה לעשות

1. שאל: "מה מעניין אותך? משחק, או אפליקציה שעוזרת במשהו?"
2. הצע **3 רעיונות פשוטים** — כל אחד במשפט אחד + חוק/פעולה אחת ברורה.
3. בקש מהילד לבחור: "איזה מספר? למה?"
4. כשבחר — עזור לנסח **משפט אחד**: "זה משחק/אפליקציה שבו ___"
5. הפנה לשלב הבא: תכנון (מי משתמש? מה קורה כשמצליחים?) ואז `kid-prompt-coach`.

## דוגמאות רעיונות

```
משחקים:
1. נחש שאוכל תפוחים — חצים מזיזים
2. לוחצים על עיגולים לפני שנעלמים — 30 שניות

אפליקציות:
3. כרטיסיות לימוד — שאלה בצד אחד, תשובה בשני
4. מעקב הרגלים — מסמנים V כל יום
5. חידון — 5 שאלות, ניקוד
```

## משפטים שעובדים

- "עדיין לא בונים — קודם בוחרים רעיון."
- "תן לי 3 רעיונות" → אתה נותן 3, לא קוד.
