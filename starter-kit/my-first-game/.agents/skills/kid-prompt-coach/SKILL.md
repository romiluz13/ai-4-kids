# Kid Prompt Coach — מפרט קצר

מנחה לבנות **מפרט** לפני קוד. לא כותב בשם הילד — שואל עד שמוכן.

## ארבעת החלקים = מפרט קצר

| חלק | שאלה |
|-----|------|
| מטרה | מה לבנות? ("זה משחק/אפליקציה שבו ___") |
| תחום | איזה קובץ? שינוי אחד? |
| הקשר | הצבעה לקובץ (ראה טבלה) |
| בדיקה | מה רואים בדפדפן כשעובד? |

## הצבעה לקובץ — לפי כלי

| כלי | איך |
|-----|-----|
| Cursor | `@game.html` |
| Claude Code | "קרא את game.html" |
| Codex / Copilot / Gemini | נתיב הקובץ בפרומפט |
| כולם | "עדכן את הקובץ בפרויקט" |

## אחרי מפרט מוכן

עזור למלא `templates/TASKS.md` — 2–4 צעדים קטנים. אז בנייה.

## דוגמאות

```
משחק: נחש, חצים, תפוחים=ניקוד, פגיעה בזנב=הפסד
אפליקציה: כרטיסיות, שאלה→תשובה בלחיצה, ספירת נכונות
בדיקה: בדפדפן רואים שזה עובד
```
