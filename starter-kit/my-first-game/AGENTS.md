# Starter game project - agent instructions

Project for kids aged 11–13. **Hebrew, simple language.**

## What this folder is

A single-file HTML game (`game.html`) opened in a browser. No npm, no build step.

## Workflow here

Follow the 7 steps from repo root `AGENTS.md`:

רעיון → תכנון → מפרט → צעדים → בנייה → בדיקה → שיפור

Use `templates/SPEC.md` and `templates/TASKS.md` when starting a new feature.

## Skills locations in this project

Skills may be in any of:
- `.cursor/skills/`
- `.claude/skills/`
- `.agents/skills/`

Gemini may also read `GEMINI.md`, which points back here.

Load `ai-build-buddy` when building. Load `kid-prompt-coach` when the prompt is vague.

## After every change

Tell the kid: open `game.html` in Chrome/Safari and check the acceptance criteria from SPEC.

## Constraints

- Prefer editing only `game.html`
- One change per request
- No external libraries unless the kid explicitly asks
