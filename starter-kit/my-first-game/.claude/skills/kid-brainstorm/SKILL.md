# Kid Brainstorm - רעיון לפני קוד

אין קוד. אין עריכת קבצים. רק שיחה קצרה בעברית פשוטה.

המטרה: הילד בוחר רעיון שהוא רוצה, לא רעיון שהסוכן דוחף.

## תהליך

1. שאל: "מה יותר בא לך - משחק, או אפליקציה שעוזרת במשהו?"
2. הצע 3 רעיונות קטנים שמתאימים לילדים בישראל.
3. בכל רעיון כתוב פעולה אחת ברורה: לוחצים, גוררים, עונים, מסמנים.
4. בקש בחירה: "איזה מספר הכי בא לך? למה?"
5. סיים במשפט מפרט אחד: "זה משחק/אפליקציה שבו ___".
6. עבור ל-`kid-prompt-coach`.

## רעיונות טובים

```text
1. חידון כיתה - עונים על 5 שאלות ומקבלים ניקוד.
2. תופסים חטיף - מזיזים סל ותופסים דברים שנופלים.
3. כרטיסיות למבחן - שאלה בצד אחד, תשובה בלחיצה.
4. מעקב הרגלים - מסמנים מים, ספורט, שיעורים או קריאה.
5. סיפור בחירות - כל כפתור משנה את המשך הסיפור.
```

## גבולות

- אל תציע רשת חברתית, כסף אמיתי, צ'אט פתוח עם זרים, או איסוף פרטים אישיים.
- אל תבחר במקום הילד.
- אל תציג 10 רעיונות. שלושה מספיקים.

## בדיקת סיום

סיים רק כשיש משפט אחד ברור:

```
אני רוצה לבנות ___ שבו ___.
```
