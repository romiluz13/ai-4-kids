# בונה עם AI — מאגר לילדים

**גילאים 11–13 · עברית · בלי ניסיון בתכנות**

## הרעיון בקצרה

אתם לא צריכים לדעת קוד. אתם צריכים **סוכן קוד** (Cursor, Claude Code, Codex, Copilot…) ואת **המאגר הזה**.

1. שבטו את המאגר  
2. פתחו אותו בסוכן  
3. כתבו **משפט אחד** — והסוכן יהפוך למנטור שלכם

ה-AI לא רק כותב קוד. הוא **מסביר**, **מנחה**, **שואל**, **מתקן**, **מלמד** — בעברית פשוטה, בקצב שלכם.

---

## התחלה — העתיקו לסוכן

```
שבטתי את המאגר ai-4-kids.
קרא את AGENTS.md והפעל skill setup-summer-school.
התקן הכול, פתח את starter-kit/my-first-game,
ודבר איתי בעברית פשוטה כמו מנטור לילד בלי ניסיון בתכנות.
```

**זהו.** הסוכן יתקין Skills, יפתח את הפרויקט, ויתחיל ללוות אתכם.

---

## מה הסוכן עושה בשבילכם (מנטור)

| הסוכן… | דוגמה |
|--------|--------|
| **מסביר** | "מה זה לולאה?" — בלי לשנות קבצים |
| **מציע רעיונות** | 3 רעיונות למשחק — לפני שכותבים קוד |
| **מנחה תהליך** | רעיון → תכנון → מפרט → בנייה → בדיקה |
| **בונה** | משנה את `game.html` לפי מה שביקשתם |
| **מלמד** | מסביר מה עשה ואיך לבדוק בדפדפן |
| **מתקן** | "קורה X, רציתי Y" — שינוי אחד בכל פעם |

**אתם** מחליטים מה לבנות. **הסוכן** עוזר ללמוד ולבנות.

---

## התהליך — 7 שלבים

```
רעיון → תכנון → מפרט → צעדים → בנייה → בדיקה → שיפור
```

| שלב | מה עושים |
|-----|----------|
| רעיון | בוחרים מה לבנות — **בלי קוד** |
| תכנון | מי משחק? מה קורה? |
| מפרט | "זה משחק שבו ___" |
| צעדים | 2–4 משימות קטנות |
| בנייה | הסוכן כותב — משימה אחת בכל פעם |
| בדיקה | `game.html` בדפדפן — אתם בודקים |
| שיפור | תיקון אחד → שוב בדיקה |

מדריך מלא: [`reference/the-full-journey.html`](reference/the-full-journey.html)
