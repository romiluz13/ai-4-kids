מה סוכן קוד יודע לעשות

שיעור 3 · ~14 דקות · Cursor, Claude Code, Codex ודומיהם

לא רק Cursor — כל סוכן קוד מציע יכולות דומות. המורה יראה חלק מהן חי על המסך.

  • פרומפט — מטרה, תחום, הקשר, בדיקה (מדריך)
  • Open Folder — פותחים תיקיית פרויקט
  • @ קבצים — מצביעים לקובץ במקום לתאר
  • יצירה ועריכת קבצים — בונה מה שביקשתם
  • בדיקת שינויים — קוראים diff לפני אישור
  • מצבים — Ask (למידה) · Plan (תכנון) · Agent (בנייה) · Debug
  • חיפוש בפרויקט — מוצא איפה משהו מוגדר
  • חיפוש ברשת — תיעוד עדכני — ואתם בודקים
  • כללי פרויקט — CLAUDE.md / rules — הוראות קבועות
  • טרמינל — מריץ פקודות כשצריך
  • סוכני משנה — מפנה משימה לעוזר, חוזר עם תשובה
