יסודות עולם הפיתוח

כרטיס חובה · 8 רעיונות · בלי זה הכול מרגיש קסם

אתם לא צריכים להיות מומחים. אבל 8 דברים האלה חייבים להיות ברורים — אחרת לא תבינו מה הסוכן עושה ואיפה המשחק שלכם.

1. קובץ (File)

דבר שנשמר במחשב. למסמך יש שם. לקוד יש שם. אם לא שמרתם — אין מה לפתוח.

2. תיקייה (Folder)

תיק שמכיל קבצים ותיקיות אחרות. מחזיקים סדר.

3. פרויקט = תיקייה אחת

כל מה שבונים יושב בתיקייה אחת. משחק אחד = תיקייה אחת. לא לפזר על שולחן העבודה.

my-first-game/   ← הפרויקט שלכם
├── game.html   ← המשחק (פותחים בדפדפן)
└── .cursor/   ← הגדרות לסוכן (אופציונלי)

4. סיומת (Extension)

החלק אחרי הנקודה בשם הקובץ:

  • game.html — דף אינטרנט / משחק בדפדפן
  • photo.jpg — תמונה
  • song.mp3 — מוזיקה

המחשב יודע מה לעשות לפי הסיומת.

5. דפדפן = המסך שלכם

Chrome, Safari, Firefox — שם רואים את מה שבניתם. בונים בקובץ → פותחים בדפדפן → בודקים.

6. סוכן קוד = עורך חכם

Cursor / Claude Code / Codex — פותחים את תיקיית הפרויקט, מדברים במילים, והסוכן יוצר ומשנה קבצים בתיקייה. אתם לא חייבים לכתוב הכול לבד.

7. באג (Bug)

משהו לא עובד כמו שציפיתם. לא כישלון — חלק מהעבודה. מתארים ל-AI מה קרה → מתקנים.

8. איפה העבודה שלי?

בתיקיית הפרויקט. אחרי השיעור — אפשר למצוא, לפתוח שוב, להמשיך לבנות.

זרימה בקצרה

  1. צרו / העתיקו תיקיית פרויקט
  2. פתחו אותה בסוכן
  3. בקשו מ-AI לבנות → נוצר קובץ
  4. פתחו את הקובץ בדפדפן
  5. מצאתם באג? תארו → תקנו → שוב
