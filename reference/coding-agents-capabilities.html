סוכני קוד — מה הם יודעים לעשות

Cursor · Claude Code · Codex · Copilot — אותו רעיון, כלים שונים

סוכן קוד = AI שמחובר לתיקיית פרויקט. אתם מדברים — הוא קורא, כותב, מחפש, ולפעמים מריץ. היום תראו חלק מזה על המסך הגדול.

0. Open Folder — איך מתחילים

פותחים את תיקיית הפרויקט — לא קובץ בודד.

Cursor: File → Open Folder · Claude Code: מתחילים מתוך התיקייה בטרמינל

מכאן הסוכן רואה את כל הקבצים ועובד בתוכם.

1. פרומפט — איך מדברים

פרומפט = ההוראה במילים רגילות.

פרומפט טוב = מטרה + תחום + הקשר + בדיקה

→ מדריך פרומפט מלא

2. מצביעים לקבצים (@)

במקום לתאר "איפה הקוד" — מצביעים לקובץ או תיקייה.

Cursor: @game.html · @my-first-game

פחות ניחושים = תוצאה מהירה ונכונה יותר.

3. מבנה קבצים

my-project/
├── game.html           ← המשחק (דפדפן)
├── README.md
├── CLAUDE.md           ← הוראות קבועות (Claude Code)
└── .cursor/skills/     ← Skills (Cursor)

פרויקט = תיקייה אחת. שמות תיקיות וקבצים = רמזים לסוכן.

4. יצירה ועריכת קבצים

הסוכן יוצר קבצים חדשים ומשנה קיימים.

רואים שינויים בעץ הקבצים — ואז בודקים בדפדפן.

5. בודקים לפני שמאשרים

הסוכן מראה מה השתנה (diff) — שורות ירוקות/אדומות.

לא ללחוץ "אישור" בלי להבין. AI טועה לפעמים.

אחרי אישור — תמיד בדיקה בדפדפן / במשחק.

6. מצבי עבודה

Ask — מסביר, לא משנה קבצים · "מה זה לולאה?"

Plan — מתכנן לפני בנייה · משימות גדולות

Agent — בונה ומשנה קבצים

Debug — עוזר לאתר באגים

זרימה: Ask ללמידה → Plan לתכנון → Agent לבנייה → בדיקה

7. חיפוש בפרויקט

הסוכן מחפש בקבצים — מוצא איפה משהו מוגדר.

שימושי כשהפרויקט גדל.

8. חיפוש באינטרנט — למה?

  • תיעוד עדכני (API, ספרייה, שגיאה)
  • מידע שלא היה באימון של הסוכן
  • לראות איך אחרים פתרו בעיה דומה

אתם עדיין בודקים — לא הכול מהרשת נכון.

9. טרמינל

הסוכן יכול להריץ פקודות (שרת, התקנת חבילה).

היום נשאר פשוט: קובץ HTML שנפתח ישירות בדפדפן.

10. Skills — SKILL.md

Skill = קובץ הוראות שהסוכן קורא כשזה רלוונטי.

מיקום (Cursor): .cursor/skills/שם/SKILL.md

יצירה: תיקייה + קובץ עם שם, תיאור, כללים.

שימוש: אוטומטי, או /שם-ה-skill בצ'אט.

→ הסבר מפורט

11. כללי פרויקט (הוראות קבועות)

קובץ שהסוכן קורא בכל שיחה — לא צריך לחזור על אותם כללים.

Claude Code: CLAUDE.md · Cursor: AGENTS.md או .cursor/rules

דוגמה: "עברית פשוטה · רק HTML · שינוי אחד בכל פעם"

12. סוכני משנה (Sub-agents)

סוכן ראשי מפנה משימה לעוזר — חוקר נושא, מחפש בקוד, עובד במקביל.

דוגמה: "תבדוק איך עושים צליל ב-HTML" — בזמן שאתם עובדים על משהו אחר.

הלולאה שלכם (מיומנות אנושית)

תאר → למד/בנה → בדוק → תקן → שפר

הסוכן עושה הרבה — אתם מחליטים, בודקים, מנחים.
