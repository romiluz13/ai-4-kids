מילון מונחים

עזר מהיר - להדפסה או לשמירה

Vibe Coding (וייב קודינג)
בניית תוכנה במילים - אתם מתארים, ה-AI כותב קוד, אתם בודקים ומשפרים.
פרויקט (Project)
תיקייה אחת שמכילה את כל מה שבונים - משחק אחד = תיקייה אחת.
דפדפן (Browser)
איפה רואים ובודקים משחקי .html - Chrome, Safari וכו׳.
סוכן קוד (Coding Agent)
AI שמחובר לתיקיית פרויקט - קורא, כותב, מחפש. Cursor, Claude Code, Codex, Copilot.
Open Folder
פותחים את תיקיית הפרויקט בסוכן - לא קובץ בודד.
Cursor (קרסר)
סוכן קוד אחד מכמה - עורך קוד עם AI מובנה.
Claude Code
סוכן קוד של Anthropic - אותו רעיון, ממשק אחר.
פרויקט פתוח
הסוכן רואה את כל הקבצים בתיקייה שפתחתם ובוחר את הנכון לבד - לא צריך לתייג.
Agent mode
הסוכן בונה ומשנה קבצים.
Ask mode
הסוכן מסביר בלי לשנות קבצים.
Plan mode
הסוכן מתכנן לפני בנייה - למשימות גדולות.
Debug mode
הסוכן עוזר לאתר באגים.
diff
השינויים שהסוכן הציע - קוראים לפני שמאשרים.
CLAUDE.md
הוראות קבועות לפרויקט (Claude Code) - כמו rules ב-Cursor.
SKILL.md
הוראות לסוכן ב-.cursor/skills/ - לא המיומנות שלכם.
סוכן משנה (Sub-agent)
עוזר שהסוכן הראשי שולח למשימה ספציפית.
תאריך עצירה (Knowledge cutoff)
התאריך שעד אליו ה-AI למד. דברים שהשתנו אחריו - הוא אולי לא יודע. לכן בודקים ברשת.
תיעוד (Documentation)
מדריך רשמי של כלי - איך מתקינים ומשתמשים. מתחילים מ-README ו-Getting Started.
קוד פתוח (Open Source)
קוד שאנשים שיתפו בחינם - אפשר ללמוד ממנו ולהשתמש. חי הרבה ב-GitHub.
GitHub
אתר שבו שומרים ומשתפים פרויקטי קוד. שם מוצאים קוד פתוח של אחרים.
פרומפט (Prompt)
ההוראה לסוכן. טוב = מטרה + תחום + בדיקה.
הוראה (Instruction)
צעד בודד שהמחשב מבצע. "הזז שמאלה", "הצג ניקוד".
משתנה (Variable)
שם לערך שיכול להשתנות: ניקוד, שם, מהירות, מספר חיים.
החלטה (Decision / If)
אם תנאי מתקיים - עשה אחד, אחרת - עשה משהו אחר.
לולאה (Loop)
חזרה על צעדים - לולאת בנייה (אתם) או לולאת קוד (במשחק).
באג (Bug)
משהו שלא עובד כמו שציפיתם. לא סוף העולם - חלק מהתהליך.
איטרציה (Iteration)
גרסה נוספת אחרי תיקון או שיפור. בונים בגרסאות, לא בפעם אחת.
לולאת בנייה
תאר → בדוק → תקן → שפר. המיומנות האנושית עם סוכן קוד.
