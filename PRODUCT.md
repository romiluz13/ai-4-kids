# Product

## Register

brand

## Users

Kids aged 11-13 with no programming background, plus parents and teachers helping them start safely. The class surface is used in a 90-minute Hebrew session where the teacher demonstrates a coding agent live and kids participate through ideas, prompts, testing, and questions. The starter kit is used later at home inside a coding agent.

## Product Purpose

Help a child become an independent builder with AI coding agents. The repo must make the agent act like a simple Hebrew mentor: ask first, explain, build one change at a time, test in the browser, and slowly give the child more responsibility.

## Brand Personality

Warm, clear, brave. It should feel like a mentor sitting beside the child, not a developer manual and not a marketing page.

## Anti-references

Avoid AI slop: generic link hubs, inflated promises, adult developer jargon, overstuffed checklists, unexplained tool claims, decorative card grids, and internal planning language leaking into public pages.

## Design Principles

1. Show the path before the archive.
2. Teach through the mentor relationship, not through tool trivia.
3. Give one clear next action on every public page.
4. Keep the class flow presenter-led and kid-participatory.
5. State tool support conservatively and verify current behavior before making claims.

## Accessibility & Inclusion

Hebrew-first, right-to-left, short sentences, large readable type, high contrast, keyboard-friendly navigation, and reduced cognitive load for absolute beginners.
